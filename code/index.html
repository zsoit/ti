OPERATORY PORÓWNANIA

> większy od
< mniejszy od
>= większy bądź równy
<= mniejszy bądź równy
== równy
!= różny od

OPERATORY PRZYPISANIA

= zwykłe przypisanie
+= dodaj i przypisz
-= odejmij i przypisz
*= pomnóż i przypisz
/= podziel i przypisz

TYPY ZMIENNYCH

int liczby całkowite
float liczby zmienno przecinkowe
double liczby rzeczywiste
char wszystkie znaki
string tylko teskt
